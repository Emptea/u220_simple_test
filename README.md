## Простой тест работы U220
0. Подключить плату по USB 3.0 к компьютеру. Подать 40 МГц размах 3 В синусоиду на вход ref in (не 10M/PPS).
1. Запустить gnuradio
```
gnuradio-companion
```
2. Открыть через File->Open `test_example.grc`.
3. Запустить.

В запущенном окне осциллограмм и спектрограмм входов RX также есть ползунки управления амплитудой и частотой подаваемой комплексной синусоиды. 

Частота излучения 3.6 МГц, усиление 60 дБ, ПП передатчика - 200 кГц, приемника - 1 МГц, частота дискретизации 5 МГц. Все параметры меняются в соответствующий полях блоков `Variable`.
